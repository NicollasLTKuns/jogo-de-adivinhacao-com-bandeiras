Jogo de Adivinhação de Bandeiras

Descrição
Este jogo apresenta ao usuário o nome de um país e quatro opções de bandeiras. Entre as quatro alternativas, apenas uma está correta.

Funcionamento
Ao selecionar uma das opções, o sistema verifica se a resposta está correta.
Se a escolha estiver correta, o fundo da bandeira selecionada ficará verde e o contador de acertos será incrementado.
Caso a escolha esteja incorreta, o fundo da bandeira ficará vermelho e o jogador perderá uma vida.
O jogador possui um total de três vidas.

Objetivo
Acertar o maior número possível de bandeiras antes de perder todas as vidas.
